# Calculadora Simples em Python

Este é um projeto simples de calculadora desenvolvido em Python, criado como parte dos meus estudos em Engenharia de Software.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão (com tratamento de erro para divisão por zero)

## Como usar
================================
CALCULADORA SIMPLES
================================
1 - Somar
2 - Subtrair
3 - Multiplicar
4 - Dividir
================================
Escolha uma opção (1-4): 1
Digite o primeiro número: 10
Digite o segundo número: 5

✅ Resultado: 10.0 + 5.0 = 15.0
================================
